gpt-oss-120b (August 2025) is OpenAI's first open-weights model release since GPT-2 in 2019.
The introduction blog post and the Hugging Face model card disclose the architecture as a
sparse Mixture-of-Experts transformer with the now-dominant open-weights backbone:

- **117B total parameters, 5.1B active per token.** 128 experts per MoE layer with top-4
  routing — a relatively wide, fine-grained routing recipe compared to Mixtral's 8-experts /
  top-2 baseline.
- **Pre-Norm RMSNorm + RoPE + GQA + SwiGLU** dense backbone.
- **Alternating attention pattern**: most layers use banded sliding-window attention; a
  subset use full (dense) attention. The interleaving keeps long-context KV-cache cost in
  check at the 131K-token window.
- **MXFP4 expert quantization** is documented as the default inference format — the expert
  weight matrices ship in the OCP Microscaling FP4 format, with the rest of the model in
  BF16/FP16.

The release is under Apache 2.0 license. Specific quantitative architecture parameters (exact
head dim, exact layer count, exact expert FFN width, RoPE base) are present in the released
model config files rather than highlighted in the announcement; this entry sticks to what
the official announcement and model card describe.
