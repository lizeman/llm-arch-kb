Llama 3.1 70B is Meta's flagship dense decoder from the Llama 3 family. Architecturally it is
the canonical "consensus stack" for dense models in 2024–25: **Pre-Norm RMSNorm + RoPE + GQA +
SwiGLU**. The 70B model has 80 layers, 64 query heads sharing 8 KV heads (GQA group count 8).

Llama 3.1's training extends context from 8K to 128K via continued pre-training with a scaled
RoPE base rather than a parameter-free interpolation scheme like YaRN.
