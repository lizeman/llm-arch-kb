Mistral 7B (September 2023) is the model that brought sliding window attention to the
production frontier and gave Apache-licensed open weights to a model competitive with much
larger dense baselines. Architecturally a 32-layer transformer with the Pre-Norm RMSNorm + RoPE
+ GQA + SwiGLU canonical stack, plus the defining addition:

- **Sliding window attention** with $W = 4096$. Each layer's attention is local; the stacked
  receptive field reaches $32 \times 4096 \approx 131K$ tokens. The recipe demonstrated that
  pure local attention is sufficient for a competitive 7B-class model. See the
  [SWA entry](/llm-arch-kb/attention/sliding-window/).

The Mistral 7B paper's empirical comparisons (Table 2) showed the model matching or beating
Llama 2 13B on most benchmarks despite half the parameters — a result mostly attributable to
data and training rather than architecture, but the architectural simplicity made the recipe
easily reproducible. Subsequent open-weights efforts (Mixtral 8x7B from the same lab, OLMo 1
from AI2) adopted variations of the same stack.

Although the v1 ship explicitly draws core adoption lists from Llama / OLMo / DeepSeek / Gemma
/ Kimi / MiniMax / GLM, Mistral 7B meets the citation bar of [§9](/llm-arch-kb/about/) — open
weights, paper-documented architecture, Apache license — and is included here as the
canonical SWA adopter.
